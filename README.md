# Badge PWNterrey 2025
Los patitos llegaron es el badge electronico de [PWNTERREY]([https://hackgdl.net/](https://pwnterrey.net/))

## Características

- Microcontrolador Puya PY32F002AA15M6TU
- Baterias CR2030
- 4 LEDs SMD
